# SCCEX

> Sugar ContChat Editor X

## 使用方法

1. 将整个压缩包解压到一个空文件夹中
2. 双击运行 SugarContChatEditor.exe

### 警告

**目前 SCCEX 在退出或新建项目时不会检查项目是否保存，请在执行这些操作前保存项目**

## 常见问题

### 启动时报错 `系统错误：由于找不到 VCRUNTIME140_1.dll，无法继续执行代码...`

原因：VC++2015 环境未安装或损坏

需要安装或修复 `Visual C++ Redistributable for Visual Studio 2015` 环境

或尝试将程序内的 `_vcruntime140_1.dll` 重命名为 `vcruntime140_1.dll` 

(如果第二种方法失败建议还原其名称或将其删除)

https://www.microsoft.com/zh-CN/download/details.aspx?id=48145

### 启动时报错 `系统错误：由于找不到 xxxxxxxxx.dll，无法继续执行代码...`

原因：此应用程序打包出现问题，缺少需要的环境

前往 https://github.com/This-is-XiaoDeng/SCCEX/actions/workflows/build.yml ，使用最新版的 Full 版本（`SCCEX for Windows (Full)`）重试
